# 天气 App

这是一个使用 Vue3 和 Tailwind 编写的天气 App。

## 功能

- 可以输入城市名，获取该城市的实时天气情况。
- 可以查看未来七天的天气预报。
- 可以切换温度单位，支持摄氏度和华氏度两种单位。

## 使用方法

1. 克隆代码仓库：

```bash
git clone https://github.com/ECarry/weather-vue.git
```

2.进入项目目录：

```bash
cd weather-app
```

3.安装依赖：

```bash
npm install
```

4.运行项目：

```bash
npm run serve
```

5.在浏览器中打开：

```javascript
http://localhost:5173/
```

## 技术栈

- Vue3
- Tailwind
- Axios
